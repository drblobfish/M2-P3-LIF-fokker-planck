# Une méthode de différences finies pour la résolution numérique de l'équation de Fokker-Planck du modèle LIF


Réplication de certains résultats de l'article :

Hu, J., Liu, J.-G., Xie, Y., Zhou, Z.: A structure preserving numerical scheme for Fokker-
Planck equations of neuron networks: numerical analysis and exploration, http://arxiv.org/abs/
1911.07619

## Structure

```
├── fig2.py               fig[2,3,4,5].py génère les figures du rapport
├── fig3.py
├── fig4.py
├── fig5.py
├── figs                  dossier contenant les figures du rapport
├── LIF.py                implémentation de la méthode numérique
├── movies                dossier contenant les animations
├── movies.py             génère les animations
├── order_accuracy.py     calcul de l'ordre de convergence de la méthode
├── rapport.typ           rapport
├── refs.bib
└── slide.typ             diaporama
```


Ce travail a été effectué dans le cadre de l'UE "Modélisation Stochastique en Neuroscience" de l'UFR de Maths de Sorbonne Université.
